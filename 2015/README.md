# LIWC Translation 2015

## How to use:
```
    Usage:
        main.py [options]

    Options:
        -h --help           Show this screen.
        -d --dict   FILE    Supply own dictionary.
        -c --corpus FOLDER  Specify test corpus [default: corpora/dpc].
```

## Requirements:

* nltk
    *   brown corpus
    *   pos_tagger
* TreeTagger
